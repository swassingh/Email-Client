# PM Vision — NovaMail (POC)

## Product Vision
NovaMail is a fast, trustworthy email client that helps professionals clear inboxes in minutes with AI-assisted prioritization, concise thread summaries, and one-click action suggestions.

## Problem
People spend too much time triaging low-value email and switching between communication and task tools.

## POC Goals (1 week)
1. Compose + inbox baseline experience.
2. Fast API-backed message retrieval.
3. Team-ready architecture and implementation plan.

## Success Criteria
- User can compose and persist emails.
- User can filter inbox by recipient.
- Architecture document is ready for manager review.
